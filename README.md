# Simulador Financeiro: De R$ 3.000 a Milionário (ou Quebrado)
#Demo: https://luisturra-jogo-de-finan-as-pessoais--simul-streamlit-app-isw5b2.streamlit.app/
Um jogo/simulador **realista** que mostra como suas escolhas diárias impactam seu patrimônio em **1, 5, 10, 15 ou 20 anos**.

---

## O que é Monte Carlo? (Explicação Simples)

> **"Monte Carlo" = jogar dados 500 vezes para prever o futuro.**

### Como funciona no simulador:

1. Você escolhe:  
   - Moradia;
   - Transporte;
   - Educação; 
   - Investe;
   - Lazer.

2. O sistema **roda 500 vidas diferentes** com:
   - Mercado sobe? Cai? (aleatório, mas realista)
   - Inflação varia?
   - Salário aumenta?

3. No final, ele mostra:
   - **Mediana** → "Onde você provavelmente vai estar"
   - **Pior 10%** → "Se der tudo errado"
   - **Melhor 10%** → "Se der tudo certo"
   - **Faixa azul** → 80% dos cenários ficam aqui

---

## Funcionalidades

- Escolha moradia, transporte, lazer, educação e investimento
- 1 clique → resultado completo
- Gráfico com **faixa de erro 80%**
- Métricas: Mediana, Média, Pior/Melhor 10%
- Download de **PDF + CSV**
- Funciona **offline**
